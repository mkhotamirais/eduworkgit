Fundamental

Git adalah salah satu version control system yang mencatat setiap perubahan file sourcecode. Bermanfaat untuk menyimpan source code, mudah berkolaborasi dengan programmer seluruh dunia dan berkontribusi pada project open source.

git adalah sebuah program/engine sedangkan github, gitlab dan bitbucket adalah layanan cloud. Tools yang dibutuhkan adalah install git, buat aku github, gitlab atau bitbucket, sourcetree untuk GUI dan text editor. Adapun perintah dasar yang sering digunakan untuk mengoperasikan git adalah:

  • git init: inisialisasi repositori yakni mengubah folder menjadi repo
  • git config: konfigurasi username dan email; global untuk server; local untuk local
    • git config --list: melihat semua config
    • git config user.name: melihat config username
    • git config user.email: melihat config email
    • git config user.name "username_lain": mengganti username
  • git status: melihat status file (untrack, staging area, modified)
  • git add .: perintah untuk memindahkan file yang baru dibuat dari untrack ke staging area
  • git commit -m "pesan": perintah untuk mencatat perubahan yang dilakukan
  • git log: melihat riwayat commit
  • git clone http://...: menduplikasi repo dari server / layanan cloud
  • git remote
  • git push: mengirim commit ke server, buat repo baru di server kosongkan isinya, pada bagian akhir, copy satu per satu perintah ke terminal
  • git pull: mengambil commit dari server jika ada perubahan di server

Branch

Branch adalah cabang atau copy-an dari repo, berikut perintah branch

  • git branch: melihat semua branch (local)
  • git branch --all: melihat semua branch (public)
  • git branch nama_branch: membuat branch baru
  • branch dapat dibuat lebih dari 2, sesuai dengan kebutuhan
  • git checkout nama_branch: berpindah branch
  • ketika kita berpindah branch, maka perubahan commit hanya akan tersimpan di branch yang kita pilih saja
  • git branch -d nama_branch: menghapus branch jika sudah tidak dibutuhkan lagi
  • git merge nama_branch: menggabungkan branch, relatif pada branch yang dipilih / aktif saat ini, pastikan semua file sudah dicommit
  • ketika ada conflict yakni ketika 2 merge memiliki perubahan pada file yang sama maka commit tidak bisa dilakukan kecuali setelah conflict diselesaikan

Git Gui (Sourcetree)

Software berbasis GUI untuk mempermudah pengoperasian git. semua perintah yang bisa dilakukan di cli dapat dilakukan dengan mudah dengan visualisasi dengan git GUI.
