Struktur data

javascript adalah bahasa pemrograman yang membuat website lebih interaktif, mudah dipelajari, fleksibel, mutiplatform

  • variable (var, let, const): tempat menampung data
  • tipe data: undefined, string, number, boolean, array, object, empty, NaN, Dynamic Type; let a, b = "abc", c = 10, d = true, e = ["satu", "dua"], f = { b: "abc", c: 10 }; console.log(typeof a, typeof b, typeof c, typeof d, typeof e, typeof f);
  • operator: + - * / %

Control flow

cara untuk manipulasi alur program

  • pengkondisian: if, else, switch, case
  • perulangan: native loop, higher order function

function

sebuah program yang bisa digunakan kembali, function = object = first class function, jenisnya: declaration, expression, arrow, constructor

Destructuring

Fitur untuk mengambil value dari array atau objek dan disimpan ke dalam variabel baru secara masif

oop

pola yang berorientasi pada objek, mengelompokkan fungsi dan var sesuai konteksnya; class itu blueprint object dan membungkus fungsi dan variable; method itu fungsi dalam objek; property itu var dalam objek; constructor itu fungsi yang berjalan otomatis saat objek dibuat; inheritance keturunan

Module System

isolasi kode dari satu file ke file lain. manfaatnya modularitas keamanan dan kerapian. hanya suport es6

asynchronous

cara eksekusi program untuk menghasilkan output tanpa melihat urutan eksekusi. javascript secara default eksekusi perintah secara synchronous, kecuali untuk beberapa hal seperti ajax, websocket, worker, file, database dll; bisa juga buat sendiri dengan setInterval atau setTimeout. teknik asynchronous digunakan biasanya dalam digunakan dalam mengelola komunikasi yang tidak mungkin singkron atau harus menunggu seperti ajax, operasi file, koneksi ke database, websocket, real time communication seperti pada app chatting dll;

callback

fungsi yang dieksekusi melalui parameter. untuk menggunakan fitur asynchronous dan percepat performa. dibutuhkan untuk event listener dan tangani asynchronous. function dalam javascript adalah object atau first-class-object yang artinya bisa dijadikan parameter, disimpan ke variabel, seperti objek pada umumnya bisa memiliki property dan method, mengembalikan value berupa fungsi;

callback disebut juga higher order function yakni fungsi yang dieksekusi melalui parameter fungsi lain
